This project performs statistical hypothesis testing using a right-tailed Z-test to evaluate whether the weekly operating costs of Bombay Hospitality Ltd. restaurants have increased beyond the company's theoretical operating cost model.

The project demonstrates:
- Hypothesis formulation
- Z-test calculation
- Critical value comparison
- Statistical decision-making
- Business interpretation using statistical evidence

This project highlights how businesses use statistics to validate operational assumptions and support evidence-based decision-making.
